Ubuntu 12.04
============

For Development
---------------

sudo apt-get install git python-dev python-virtualenv libxml2-dev libxslt1-dev libjpeg62 libjpeg-dev libfreetype6 libfreetype6-dev zlib1g-dev gdal-bin postgresql-9.1-postgis proj

Fix missing symlinks on 64bit Ubuntu:
    ln -s /usr/lib/x86_64-linux-gnu/libjpeg.so /usr/lib
    ln -s /usr/lib/x86_64-linux-gnu/libfreetype.so /usr/lib
    ln -s /usr/lib/x86_64-linux-gnu/libz.so /usr/lib

$ git clone https://github.com/friendlycode/viget-lw.git
$ cd viget-lw
$ make install_dev
$ make setup_dev
$ make run_server_dev
$ ^C
$ make stop_solr

Notes:
------

Solr runs on port 8080. You can access the admin at http://localhost:8080/solr/admin/
To shut down solr: `make stop_solr`
